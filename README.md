# AI-Powered Decision Intelligence System
## Project Overview
This project predicts customer churn using Machine Learning and explains predictions using Explainable AI (SHAP).

## Technologies Used
- Python
- Pandas
- Scikit-learn
- SHAP
- Jupyter Notebook

## Features
- Customer churn prediction
- Explainable AI
- Business insights
- AI recommendations

## Model Used
Random Forest Classifier
